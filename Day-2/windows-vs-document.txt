Windows Object
--------------
It is the top most object and outermost element of the object hierarchy.

The window object represents a window/tab containing a DOM document.

window object have properties and methods.

Document Object
----------------
Each HTML document that gets loaded into a window becomes a document object.

The document object is the property of the window object that points to the DOM document loaded in that window.

Document object also have properties and methods.

When a web page is loaded, the browser creates a Document Object Model of the page.

The document object represents the whole html document such as HTML, head, body, html tags etc.
